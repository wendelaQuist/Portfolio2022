Swimming duck

For development we got an example which contained moving an object with JavaScript. Tessa and I both worked on our duck but are creating different POC’s. I made my duck interactive by programming it to move when you press the “W”, “A”, “S”, “D” keys. At first I made a canvas within my screen so my duck doesn't run off. I gave the duck a start position so it won't start on the top of the navigation bar. When the keys are pressed it will look at the position of the duck. When the duck is against the border it will stop going in that direction. I gave the duck a 20px pace so it will add or remove 20px to the last position.
