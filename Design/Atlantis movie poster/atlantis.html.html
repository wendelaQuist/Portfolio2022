Movie poster

The style of atlantis is kind off similar to tribal patterns, so I wanted to combine this style with neon glow.

I realized for a minimalistic poster the patterns are a bit overcrowded, so I changed it to simple line art with a glow effect.

My plan was to focus solely on Kida. For me she was the main character because she was powerful and a leader.

Reflection

If I worked on this project with photoshop sooner instead of Illustrator, I could have probably spend less time.

I learned how to create a realistic glow by using filters. I got some feedback and seeing how small Kida is compared to the rest of the poster,

I want to only use Kida's head in the beam of light so I would need to redraw the middle part.
